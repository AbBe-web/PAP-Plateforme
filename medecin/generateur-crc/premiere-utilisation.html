📘 PAP — Démarrage rapide

PAP est une aide cognitive structurée pour la consultation d’activité physique. L’outil aide à structurer l’évaluation, repérer les points de vigilance et générer un compte-rendu réutilisable.

▶ 1. Commencer une consultation

SI le patient a rempli le questionnaire PAP :

  • Scanner le QR code patient
  • OU importer les données patient

SINON :

  • Utiliser « Nouvelle consultation »

🩺 2. Pendant la consultation

Le formulaire aide à :

  • Structurer l’évaluation
  • Repérer les points de vigilance
  • Adapter les conseils d’activité physique
  • Générer un compte-rendu réutilisable

Les propositions affichées sont basées sur les référentiels HAS et restent à interpréter selon le contexte clinique.

Le médecin reste décisionnaire final.

🧠 3. Aide contextuelle

Selon les données saisies, PAP peut afficher :

  • Précautions
  • Adaptations
  • Repères pathologies
  • Éléments de vigilance
  • Messages patient possibles

Le bouton « Actualiser l’analyse » met à jour les éléments affichés.

📝 4. Générer le compte-rendu

Formats disponibles :

  • Courrier complet
  • Dossier médical
  • Paragraphe synthétique
  • Version patient

Le contenu peut être :

  • Modifié
  • Copié
  • Imprimé
  • Intégré au logiciel métier

🔒 5. Confidentialité

  • Données conservées localement sur l’appareil
  • Aucun compte requis
  • Aucun envoi automatique vers un serveur distant

⚠ 6. Important

  • PAP est une aide cognitive structurée
  • Sans automatisation décisionnelle
  • Non substitutive au raisonnement clinique

Le médecin reste décisionnaire final.
